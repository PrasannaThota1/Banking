# Banking App (Scaffold)\n\nThis workspace contains a scaffold for a banking application:\n- `backend/` - FastAPI backend scaffold with basic auth, accounts, transactions\n- `frontend/` - Angular frontend skeleton and run instructions\n\nNext steps:\n1. Create a Python virtualenv and install backend dependencies from `backend/requirements.txt`.\n2. Configure `.env` with `DATABASE_URL` and `SECRET_KEY`.\n3. Initialize DB and run Alembic migrations (or create tables).\n4. Start backend: `uvicorn app.main:app --reload --port 8000`\n5. Scaffold full Angular app (see `frontend/README.md`) and run the frontend.\n"# Banking"
